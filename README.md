## 🚀 Getting Started

    git clone https://github.com/xa1nu/faq-telegram-bot
    cd faq-telegram-bot
    pip install -r requirements.txt

Create `.env` file:

    BOT_TOKEN=your_telegram_bot_token

Run:

    python bot.py
