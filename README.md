## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/xa1nu/faq-telegram-bot
cd faq-telegram-bot
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Create .env file
```
BOT_TOKEN=your_telegram_bot_token
```

### 4. Run the bot
```bash
python bot.py
```
