# 🤖 LinguaFlow FAQ Bot

Telegram FAQ bot for LinguaFlow — online English school.
Answers student questions instantly, 24/7.

> ⚠️ This is a demo project. Most information is fictional.

## 🚀 Bot is live!
👉 @LinguaFlow_FAQ_Bot
🟢 Running 24/7 on Railway

📅 Class schedule
💳 Payment & pricing
📚 Course levels
📝 Homework submission
🏆 Certificates
🎁 Free trial lesson
👩‍🏫 Teachers info
⭐️ Student reviews
👥 Learning formats

- Python 3.13
- pyTelegramBotAPI
- python-dotenv
- Deployed on Railway
